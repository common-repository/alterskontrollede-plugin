== Description ==

Das Plugin bietet die Möglichkeit bestimmte Inhalte auf Wunsch nur über eine Altersverifikation anzeigen zu lassen.

Es ist möglich einen ganzen Beitrag/Seite zu schützen, oder es kann durch einen Shortcode auch nur ein bestimmter Bereich in einem Beitrag/Seite geschütz werden.
Dem Nutzer ist es über den Editor möglich diese Shortcodes ganz einfach zu nutzen (Screenshots 2.).

Das Tor wird entweder im Beitrag/Seite angezeigt, oder es ist möglich ein Widget zu nutzen, dieses lässt sich einfach und simpel in einer Sidebar platzieren. So erhält man ein fest plaziertes Loginforumlar.

== Frequently Asked Questions ==
Q: Wie aktiviere ich das Plugin für einen Beitrag oder eine Seite?

A: Gehe auf Beitrag/Seite  und wähle den/die zu schützenden Beitrag/Seite  aus, Du solltest eine neue Metabox (Alterskontrolle / Inhalt Schützen) sehen, setze den Haken bei "Schützen mit alterskontrolle.de".

Q: Woher bekomme ich Alterskontrolle API-key und Webmaster ID (wid)?

A: Klicke unter Einstellungen / Alterskontrolle auf "Melde Dich hier an", klicke auf Anmelden und Registriere Dich. Nun bekommst du deine Webmaster ID. Jetzt kannst du auf den Link "Alterskontrolle.de API Seite" klicken, dort findest du den API-key

== Installation ==

1. Kopiere das Plugin in den Plugin ordner.
2. Aktiviere das Plugin.
3. Nun lässt sich das Plugin unter Einstellungen / Alterskontrolle anpassen.
